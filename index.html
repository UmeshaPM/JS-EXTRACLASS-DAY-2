Q2. In how many ways can we declare a variable?

  • We can declare variables in three ways:

    • 1.Var
    2.Let
    3.Const

Q3. Difference between var,let and const?

  • The difference between Var, Let and Const are:

    • 1.Var :
    It is global scoped.
    It can be redeclared.
    Var can be modified later.

    2.Let :
    It is block scoped.
    It can't be redeclared.
    Let can be modified later.

    3.Const :
    It is block scoped.
    It can't be redeclared.
    Const can't be modified later.

Q4. What is Hoisting?

  • Hoisting :Hoisting is a Javascript Mechanism where variables and function declarations are moved to top of their scope before the code execution.
    Note :Only Declarations are hoisted not the initilization
  • Only "Var" supports hoisting.

Q5. What is an temperal dead zone?

  • It is the area of a block where a variable is inaccessible until the moment computer initilizes it with a value
  • TDZ occurs when we declare a variable of Let and Const keyword.

Q6. What is the Difference between == & === Operator ?

  • == only checks the value
  • === checks the both value and the data type
